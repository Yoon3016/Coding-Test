문제
배열을 정렬하는 것은 쉽다. 수가 주어지면, 그 수의 각 자리수를 내림차순으로 정렬해보자.

입력
첫째 줄에 정렬하려고 하는 수 N이 주어진다. N은 1,000,000,000보다 작거나 같은 자연수이다.

출력
첫째 줄에 자리수를 내림차순으로 정렬한 수를 출력한다.

예제 입력 1 
2143
예제 출력 1 
4321

예제 입력 2 
999998999
예제 출력 2 
999999998

-------------------

#include <stdio.h>

int main()
{
	int N;
	int arr[10] = { 0 };
	scanf("%d", &N);
	while (N >= 10)
	{
		arr[N % 10]++;
		N /= 10;
	}
	arr[N]++;

	for (int i = 9; i >= 0; i--)
	{
		if (arr[i] != 0) {
			for (int j = 0; j < arr[i]; j++) {
				printf("%d", i);
			}
		}
	}
	return 0;
}
